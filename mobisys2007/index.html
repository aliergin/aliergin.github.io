MOBISYS'07 Traces and Downloads

In this page, files that are used in our Mobisys 2007 paper are made available for public use. Some of the files might be very large, please be patient and use utilities that support resuming for http downloads. Please direct questions regarding the files distributed herein to Mesut Ali Ergin at ergin@winlab.rutgers.edu .Thanks for your interest.

  • User arrival pattern file (sta_arrivals.tar.gz)
    This file has relative delays of the station arrivals in seconds. A control loop like
    for (i=1;i<352;i++)
    {
    create_station(i);
    sleep( read_line(i, sta_arrivals) );
    }
    will create stations similar to the end of lunch time in 64nd IETF conference.
  • WINLAB 2006 Application Characterization capture trace (WINLAB_capture.tar.gz)
    This file is an Ethereal capture with 256 byte truncation from the main lobby AP of WINLAB (Cisco 1200) operating at Ch.6
